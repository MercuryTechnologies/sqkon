# Getting Started

Add Sqkon to your project, create your first store, and learn what each platform
needs to wire up. The pages in this section are intended to be read in order if
you're new — but each one stands alone if you only need a refresher.

1. [Installation]({{ '/getting-started/installation/' | relative_url }}) — add the dependency from Maven Central.
2. [Quickstart]({{ '/getting-started/quickstart/' | relative_url }}) — define a model, create a store, insert and query in under five minutes.
3. [Platform setup]({{ '/getting-started/platform-setup/' | relative_url }}) — Android, JVM, threading, and database location details.
4. [Serialization]({{ '/getting-started/serialization/' | relative_url }}) — the recommended `Json` defaults and how to customize them.
